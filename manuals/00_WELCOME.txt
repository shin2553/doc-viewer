Welcome to Smart Viewer v1.1.3!
If you can read this, the viewer (and text file loading) is working correctly.
This file is located in the 'manuals' folder.
You can put your PDF, Image, Log, and Code files in this folder to view them.
Supported formats:
- Documents: PDF, MD, TXT, CSV
- Images: PNG, JPG, WEBP, GIF
- Code: PY, JS, HTML, CSS, JAVA, C, CPP, H, RS, GO, XML, YAML, JSON, BAT, SH, SQL, PROPERTIES, GRADLE, DOCKERFILE, GITIGNORE
- Media: MP4, WEBM
